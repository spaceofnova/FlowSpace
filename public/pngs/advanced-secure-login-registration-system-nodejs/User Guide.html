Requirements

I recommend you read the following tutorials on our website (if you haven't already):

  1. https://codeshack.io/basic-login-system-nodejs-express-mysql/

You will need the following:

  1. Node.js
  2. MySQL Server 5.6 or higher - Make sure the port is 3306 (default).
  3. MySQL Workbench
  4. A code editor; You can use notepad to edit the Python files, but I don't recommend it. Instead, download one of the following: Notepad++, Visual Studio Code, or Atom.

Instructions

Open the command line as administrator and navigate to your advanced directory (cd c:\advanced_package_source\, etc.)

Install the below Node.js packages via command line.

  1. npm install -g nodemon --save
  2. npm install nunjucks --save
  3. npm install express --save
  4. npm install express-session --save
  5. npm install mysql --save
  6. npm install nodemailer --save
  7. npm install uuid --save
  8. npm install node-fetch@2 --save

Execute the "nodelogin.sql" file in MySQL Workbench. The SQL file includes a member and admin account that you can use for testing purposes.

If you're using Windows, you can click the "Start_Server_(Windows).bat" file that is included. If not, execute the following in command line: nodemon --use_strict main.js

If you've successfully started the server, you can navigate to http://localhost:3000/ in your browser. It should take you to the login form.

FAQs

  1. Why am I not receiving the activation email?
    You need a working SMTP mail server and make sure your firewall is not blocking ports 25, 587, 465, 110, etc. This shouldn't be a problem on most hosting providers or Linux servers (built-in email server).
  2. Why am I receiving a MySQL error?
    Make sure you update the MySQL details in the "main.js" file. You might need to change the hostname if yours is different and/or port number, and make sure you've executed the correct SQL file in MySQL Workbench.
